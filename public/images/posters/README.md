# Poster Images

Place fallback poster images for 3D scenes here. These are shown when:
- 3D rendering is not supported
- User has prefers-reduced-motion enabled
- Device is low-end

## Required Files

- `condo-poster.jpg` - Condo tower lobby scene fallback
- `medical-poster.jpg` - Medical clinic scene fallback
- `restaurant-poster.jpg` - Restaurant scene fallback
- `warehouse-poster.jpg` - Warehouse scene fallback
- `retail-poster.jpg` - Retail store scene fallback
- `office-poster.jpg` - Office suite scene fallback
- `default-poster.jpg` - Default fallback

## Recommended Specs

- Format: JPG or WebP
- Dimensions: 1200x1200 (1:1 aspect ratio)
- File size: < 200KB
- Style: Match the 3D scene appearance
